# Glossary — book-specific meanings

One line each. Terms whose meaning in this book is **load-bearing** and not generic. Full definitional essays live in `~/writing/relational-musicality/glossary.md`; this file is for the agent to anchor on quickly.

- **relational musicality**: the human capacity to exchange information through time in ways that transform participants by coordinating attention, action, feeling, and meaning. Not musical skill or taste.
- **technology of remembering**: a practice that restores integration where civilizational specialization has separated things (e.g., mind from body, self from other).
- **dismemberment**: what specialization does — separating mind/body, self/other, art/therapy/governance into bounded domains.
- **tyranny of sequential cognition (TSC)**: the structural condition in which sequential processing is treated as the only valid mode of intelligence.
- **the three orientations**: outward (computation), inward (contemplation), between (relationality). Not three intelligences; not exhaustive — "at least three."
- **computation**: specifically, what a machine can replicate. We called *people* "computers" before we built machines to do it.
- **relational computation**: distributed processing that occurs *between* bodies in interaction — not metaphor; literally takes inputs, holds state, produces outputs neither participant could alone.
- **musical relationality**: the active condition of being in mutual transformational exchange through music and movement in shared time.
- **mutual transformational exchange**: working description of what happens in these practices; two or more people transforming each other in real time.
- **choreo-musical system**: a practice in which movement and music are mutually constitutive — neither subordinate to the other.
- **relational container**: an environment whose constraints make transformation likely; not a "safe space" in the comfort sense.
- **generative constraint**: a limitation that enables creativity rather than stifling it. Repetition + constraint = freedom.
- **repair**: the capacity to return to coherence after error. The signature of a resilient relational system.
- **participation gradient**: the range from peripheral observation to full immersion. On-ramps for novices; ladders for practitioners.
- **roda**: the capoeira circle; here, the shared space that constitutes the group.
- **berimbau / gunga**: stringed bow instrument; the gunga is the lead berimbau that governs the capoeira game.
- **clave**: the rhythmic pattern that organizes Afro-Cuban music; functions as a shared reference, not a foreground beat.
- **piquetes / repiques**: in bomba, dancer's body movements that the drummer must follow — inverting the Western assumption that sound leads, body follows.
- **RALF**: Relational Audio-visual Learning Framework. The author's motion-to-sound system. Thesis in code.
- **co-regulated music**: sound environments that listen to the relational state of bodies and shape that state in return.
- **agonistic reappropriation**: taking tools built for the bounded self (computation, AI) and redirecting them toward relational ends.
